Платежные функции

  1. Автоплатежи

    Полное управление автоплатежами через API или в личном кабинете ArsenalPay. Ваши клиенты смогут самостоятельно изменить все настройки автоплатежа, без обращения в поддержку.

    Автоматическое изменение суммы автоплатежа при смене тарифа.

    Удаление по просьбе пользователя, или при расторжении договора.

    Временное отключение автоплатежа и изменение периодичности.

  2. Сохранение карты

    Позволяет совершать оплату в 1 клик. Увеличивает процент успешных платежей. Уже 40% пользователей сохраняют свои карты, и доверие с каждым годом только растет.

  3. Оплата без 3DS

    Функция позволяет настраивать лимиты на сумму платежа, при которых можно не запрашивать код подтверждения оплаты 3DS. Пользователям не нужно вводить 3DS, оплата становится быстрее и удобнее. Количество успешных платежей увеличивается.

  4. Страница при нуле

    Виджет оплаты можно легко встроить на страницу, которая информирует о нулевом балансе. Клиенты смогут оплачивать услуги без дополнительной авторизации даже при отрицательном балансе. Если у вашего сервиса нет такой страницы, мы сделаем ее для вас.

    Мы предоставляем API для получения актуального списка IP-адресов 3DS страниц банков. Что позволяет совершать оплату даже при закрытом доступе в интернет.
